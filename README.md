# ISO 27001 Risk Register – GRC Portfolio Project

This project demonstrates a basic Governance, Risk, and Compliance (GRC) risk assessment aligned with ISO 27001 principles.

## Scenario

A simulated fintech company, Stratford Pay Solutions Ltd, was used to demonstrate how organisational risks can be identified, analysed, and documented.

## Project Scope

The project includes:

- Identification of organisational assets
- Threat and vulnerability analysis
- Risk scoring using likelihood and impact
- Recommended security controls
- Mapping risks to ISO 27001 control areas

## Tools Used

- Microsoft Excel
- ISO/IEC 27001 risk management principles
